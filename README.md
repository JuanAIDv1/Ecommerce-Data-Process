Project Overview
This project demonstrates a complete end-to-end data analytics workflow
applied to an e-commerce use case.

The objective is to show the full process:
data extraction, processing, analysis and business-oriented visualization.

Project Workflow
1. Web scraping to extract raw e-commerce data
2. Data cleaning and transformation
3. Exploratory data analysis
4. Dashboard creation in Power BI
5. Business insights generation

Tools & Technologies
- Python (Pandas, Requests, BeautifulSoup)
- Jupyter Notebook
- Power BI
- CSV / Excel
- GitHub

Data Source
The data is obtained through a custom Python web scraper.
The extracted dataset includes product, pricing and sales-related information.
